# meowna
Uma aplicação web propositalmente vulnerável para você ownar

Passos para Instalação:

1° -> apt install apache2
2° -> apt install php
3° -> cd /var/www/html
4° -> mv /home/user/Downloads/meowna.zip .
5° -> unzip meowna.zip
6° -> cd meOwna
7° -> service apache2 start
8° -> acesse http://localhost
